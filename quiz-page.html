Q1. Which of the following is true about Javascript?

Multi-Paradigm Language Low Level Language Class Based Language None of the Above
The answer to this question is choice A. Javascript is a high-level, just-in-time compiled, and multi-paradigm programming language the conforms to the ECMAScript specifications.

Q2. Which of the following values for "x" will make the console.log value be "true" for the code below?

foo obj Window undefined
The answer to this question is choice B. The 'this' inside foo is being used to refer to the object of its instance when it is called upon, in this case 'obj' itself.

Q3. Which of the following function of Array object adds and/or removes elements in an array?

unshift() append() sort() splice()
The answer to this question is choice D. "splice()" can be used to both add and remove elements in an array.

Q4. Which HTML element is JavaScript placed in?

js script style javascript
The answer to this question is choice B. Javascript code should be placed inside the "script" element in HTML.

Q5. Which of the following is the correct syntax to write an "if" condition in JavaScript?

if: i = 5 if i = 5 if (i == 5) None of the Above
The answer to this question is choice C. The correct syntax in JavaScript to write an "if" condition is to write "if", followed by a boolean condition expressed inside round brackets "( )"
